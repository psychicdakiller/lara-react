# lara-react
Building Laravel with React JS

# Installation

+ composer install

+ npm install

+ npm run dev

+ php artisan serve

# Twilo SMS Api

+ "twilio/sdk": "^7.11"

# New env values
+ TWILIO_SID=your twilo sid
+ TWILIO_AUTH_TOKEN=your twilo auth token
+ TWILIO_PHONE_NUMBER=your twilo phone number
